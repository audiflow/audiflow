## Package Design Decisions

### Layer Placement

```
audiflow_app → audiflow_ui → audiflow_domain → audiflow_search
                                                     ↓
                                                audiflow_core
```

### JSON Parsing Strategy

| Current | Migrated |
|---------|----------|
| Manual `fromJson` | `@freezed` + `@JsonSerializable` |
| Manual `copyWith` | Generated by freezed |
| Manual validation | Validation in factory constructors |

Example - Podcast Model:
```dart
@freezed
class Podcast with _$Podcast {
  const factory Podcast({
    required String id,
    required String name,
    String? artworkUrl,
    String? feedUrl,
    String? artistName,
    @Default(<String>[]) List<String> genres,
    @Default(false) bool explicit,
  }) = _Podcast;

  factory Podcast.fromJson(Map<String, dynamic> json) =>
      _podcastFromJson(json);
}
```
