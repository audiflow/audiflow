# Interaction Guidelines

* **User Persona:** Senior/expert Flutter developer. Minimal explanations.
* **Explanations:** Provide code without explanations unless requested.
* **Clarification:** Ask for clarification only when requirements are ambiguous or critically unclear.
* **Dependencies:** Suggest appropriate packages without explanatory justification.
* **Tooling:**
  - Use `dart_format` for formatting
  - Use `dart_fix` for automated fixes
  - Use `analyze_files` for linting
