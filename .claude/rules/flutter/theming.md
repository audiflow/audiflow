# Visual Design

* **Background:** Apply subtle noise texture to the main background for a premium, tactile feel.
* **Shadows:** Multi-layered drop shadows for depth; cards have soft, deep shadows to look "lifted."
* **Interactive Elements:** Buttons, checkboxes, sliders, and other interactive elements use shadows with color to create a "glow" effect.

## Project Palette

* Primary: #0D47A1 (Dark Blue)
* Secondary: #1976D2 (Medium Blue)
* Accent: #FFC107 (Amber)
* Neutral/Text: #212121 (Almost Black)
* Background: #FEFEFE (Almost White)
